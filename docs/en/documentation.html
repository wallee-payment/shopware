1Prerequisites

If you don’t already have one, create a wallee account.

2Installation

  1. Upload the plugin files to your store’s root directory using FTP/SSH or you simply install the plugin directly from the Shopware plugin store.

  2. Login to the backend of your Shopware store.

  3. Start the store’s Plugin Manager (Configuration → Plugin Manager) and open the part Installed. Click the plugin’s Install button to install it.

  4. Activate the wallee Payment plugin from the Plugin Manager.

  5. Reload the page.

3Configuration

  1. Open the settings of the wallee Payment plugin.

  2. Enter the wallee space ID and application user’s access information that you created before.

  3. Save the values by clicking on the Save button.

  4. Click on "Synchronize" to create the payment methods in the Shopware store and set up the necessary webhooks.

  5. Be aware that you link the payment methods to the shipping method, otherwise they can not be used in the checkout.

4Payment Method Configuration

Most of the payment method configuration is done in wallee. After following the previous steps, the wallee payment method configurations will be synchronized automatically into the Shopware store. There are just a few payment method settings in the Shopware store in Configuration → Payment methods.

5FAQ

5.1How can I make the payment methods appear in the checkout?

Make sure that you synchronized the plugin’s settings by pressing the "Synchronize" button.

Check that the payment methods are linked to at least one shipping method.

If this does not solve the problem, it could be that you use a special fee or coupon module that we do not support. Try to disable this plugin and see if it helps. The payment methods are only displayed if the plugin’s total calculation matches the actual order total.

5.2How can I change the look and feel, payment method name, emails etc.?

All those configurations are done directly in wallee. For more information have a look at the wallee documentation.

6Support
